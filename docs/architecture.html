sequenceDiagram participant U as 用户 participant FE as React participant CMD as Tauri Command participant SVC as kline_sync_service participant PY as Python participant TF as TickFlow participant DUCK as DuckDB U->>FE: 点击"一键补齐" FE->>CMD: invoke sync_kline('', 'incremental', 'incomplete') CMD->>SVC: run_sync_script() SVC->>PY: spawn python3 sync_kline.py PY->>TF: TickFlow.klines.batch(symbols, period, count=40) TF-->>PY: K线 JSON PY->>PY: enrich → Parquet (.data/klines/current/) PY-->>SVC: stdout JSONL 进度事件 CMD->>SVC: import_and_finalize() SVC->>DUCK: INSERT securities FROM Parquet SVC->>DUCK: INSERT kline_bars FROM Parquet SVC->>DUCK: INSERT trade_calendar SVC-->>CMD: KlineSyncResult CMD-->>FE: invalidate queries
